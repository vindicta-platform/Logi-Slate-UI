# Logi-Slate UI

**The tactical command center for competitive Warhammer 40k.**

A modern React/TypeScript interface for army management, match tracking, and meta analysis.

## Features

- **Premium Design** — Dark mode, glassmorphism, micro-animations
- **Mobile-First** — Touch-optimized for tablets
- **Real-Time** — Live updates during matches

## Quick Start

```bash
git clone https://github.com/vindicta-platform/Logi-Slate-UI.git
cd Logi-Slate-UI
npm install
npm run dev
```

Open http://localhost:3000
